## A 梯度下降法
这部分内容是介绍梯度下降, 在NN中用到最多的是SGD, 为什么不介绍SGD?

SGD, S来自与样本的随机. DL中样本很多, 通常会分Batch, 每个Batch刷的过程就是SGD, 实际上数据量小的时候, SGD和GD一样. 所以数据的Shuffle就很重要.
